Musique

  • !join pour rejoindre votre channel vocal
  • !stop pour quitter le channel vocal
  • !radio nom_de_la_radio pour jouer une radio. Liste des radios disponibles avec !radio list
  • !play https://www.youtube.com/watch?v=XXXXXXXX ou nom de la musique pour jouer une vidéo YouTube.
  • !lyrics nom de la musique pour afficher les paroles de la musique
  • !musicinfo pour afficher des informations de base sur la musique

Api Clash Of Clans

  • !clan #RJOQR2Q8 affiche les infomations sur un clan
  • !savetag #JQUPPUCG pour sauvegarder votre tag Clash Of Clans dans la base de donnée
  • !joueur #JQUPPUCG affiche les informations sur un joueur
  • !joueur @utilisateur#0000 pour afficher les informations sur un utilisateur si il a sauvegardé son tag
  • !topfr pour afficher les 10 meilleurs joueurs français
  • !topclansfr pour afficher les 10 meilleurs clans français
  • !topbe pour afficher les 10 meilleurs joueurs belges
  • !topclansbe pour afficher les 10 meilleurs clans belges

Api Clash Royale

  • !clancr #Tag pour afficher les informations sur un clan Clash Royale
  • !joueurcr #Tag pour afficher les informations sur un clan Clash Royale
  • !joueurcr @utilisateur#0000 pour afficher les informations sur un joueur si il a sauvegardé son tag
  • !savecrtag #tag pour sauvegarder son tag Clash Royale dans la base de donnée du Bot

Infos du bot

  • !annonce pour afficher le nom du channel dans lequel les messages de bienvenue sont.
  • !autorole pour afficher le nom du rôle ajouté automatiquement
  • !testjoin pour simuler un membre rejoignant le serveur
  • !testleave pour simuler un membre quittant le serveur

Configuration

  • !setprefix préfixe pour changer le préfixe du bot. Les commandes seront donc disponibles via ce nouveau préfixe et non le préfixe !. Pour le moment, le préfixe ne peut faire qu'un seul caractère. Les commandes !help et !setprefix sont quand accessibles via le préfixe !.
  • !setrejoindre #channel pour créer un rejoindre (voir ici)
  • !setannonce on/off pour activer/désactiver le message de bienvenue et d'au revoir dans le channel désiré.
  • !setautorole nom du rôle / off pour activer un rôle automatique que le rôle aura dès qu'il rejoint le serveur.
  • !setbotrole nom du rôle / off même principe que l'autorole mais pour les bots
  • !disable nomDeLaCommande pour désactiver une commande sur le channel, !enable nomDeLaCommande pour la réactiver.
  • !disabled pour voir la liste des commandes désactivées
  • !setjoinmessage pour modifier le message de bienvenue quand quelqu'un rejoint le serveur. {usermention} pour mentionner le nouveau membre, {name} pour afficher le pseudo du membre, et {membercount} pour afficher le nombre de membres dans le serveur.
