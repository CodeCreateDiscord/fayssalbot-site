Fayssalbot

Ce bot discord a été créé par FayssalZ. Il intègre de nombreuses fonctionnalités utiles comme la radio ou les données des jeux Clash Of Clans et Clash Royale.

Liste des commandes

Pour ces commandes, le préfixe ! est utilisé à titre d'exemple mais sachez que le préfixe est personnalisable via la commande !setprefix (voir plus bas).

Vous pouvez utiliser la commande !help à tout moment et avec le préfixe ! ou votre préfixe personnalisé.

Modération

  • !kick + mention pour expulser un membre
  • !ban + mention pour bannir un membre
  • !clear + nombre pour effacer un certain nombre de message entre 2 et 100
  • !addrole @utilisateur#0000 Nom du role pour ajouter un rôle à un utilisateur
  • !remrole @utilisateur#0000 Nom du role pour enlever un rôle à un utilisateur
  • !addcom NomDeLaCommande Texte de la commande pour ajouter une commande personnalisée
  • !delcom NomDeLaCommande pour supprimer une commande personnalisée
  • !listcom pour avoir la liste des commandes
  • !mute @utilisateur#0000 pour rendre quelqu'un muet, il ne peut plus parler dans le channel.
  • !unmute @utilisateur#0000 pour re-permettre à l'utilisateur de parler
  • !move @utilisateur#0000 nom du channel vocal pour déplacer un membre dans un salon vocal
  • !roleall nom du rôle pour mettre un rôle à tout le monde.
  • !setnick @utilisateur#0000 nouveau pseudo pour changer le pseudo d'un membre, mettez aucun en nom de rôle pour rétablir le pseudo par defaut.

Utiles

  • !meteo + nom de ville pour afficher la météo
  • !serveur-info pour obtenir des informations sur le serveur
  • !avatar + mention pour afficher l'avatar d'un membre du serveur
  • !uptext + message pour uploader un texte
  • !short http://www.site_a_reduire.com/ pour réduire l'url d'un site
  • !definition + mot pour obtenir la definition de ce mot
  • !google + ce que vous voulez rechercher pour faire une recherche google (à venir)
  • !youtube + ce que vous voulez rechercher pour faire une recherche youtube
  • !suggestion + message pour suggérer une amélioration du bot au développeur
  • !support + message pour signaler un problème au développeur ou lui demander de l'aide.
  • !news pour avoir des informations sur la derniere mise à jour du bot
  • !translate langue texte pour traduire un texte dans la langue désirée, liste des langues ici.
  • !rejoindre Nom du channel vocal pour rejoindre un channel si le rejoindre a été créé par un admin.
  • !botstats pour avoir des informations sur le Bot
  • !userinfo + mention/id/pseudo pour avoit des informations sur un utilisateur
  • !todolist pour sauvegarder des informations quelconques, utilisation !todolist add "Nom De L'élément" contenu de l'élément pour ajouter un élément, !todolist del "Nom de l'élément" pour supprimer un élément, et simplement !todolist puis suivre les indications pour afficher un élément.

Infos du bot

  • !annonce pour afficher le nom du channel dans lequel les messages de bienvenue sont.
  • !autorole pour afficher le nom du rôle ajouté automatiquement
  • !testjoin pour simuler un membre rejoignant le serveur
  • !testleave pour simuler un membre quittant le serveur

Configuration

  • !setprefix préfixe pour changer le préfixe du bot. Les commandes seront donc disponibles via ce nouveau préfixe et non le préfixe !. Pour le moment, le préfixe ne peut faire qu'un seul caractère. Les commandes !help et !setprefix sont quand accessibles via le préfixe !.
  • !setrejoindre #channel pour créer un rejoindre (voir ici)
  • !setannonce on/off pour activer/désactiver le message de bienvenue et d'au revoir dans le channel désiré.
  • !setautorole nom du rôle / off pour activer un rôle automatique que le rôle aura dès qu'il rejoint le serveur.
  • !setbotrole nom du rôle / off même principe que l'autorole mais pour les bots
  • !disable nomDeLaCommande pour désactiver une commande sur le channel, !enable nomDeLaCommande pour la réactiver.
  • !disabled pour voir la liste des commandes désactivées
  • !setjoinmessage pour modifier le message de bienvenue quand quelqu'un rejoint le serveur. {usermention} pour mentionner le nouveau membre, {name} pour afficher le pseudo du membre, et {membercount} pour afficher le nombre de membres dans le serveur.

Informations supplémentaires

Invitation

Pour inviter le bot sur votre serveur, veuillez cliquer ici, et accordez lui toutes les permissions cochées, cela lui permettra de fonctionner correctement.

Dons

Afin d'améliorer le bot (temps de réaction, qualité du son, ...), les dons sont vivement appréciés, toutes les infos sont disponibles via la commande !don
